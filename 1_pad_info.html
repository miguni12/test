    자궁경부암 발병률 증가

    03

    전체 여성의 자궁경부암 발병은 줄어드는 반면
    2030세대는 각각 11.9%와 14.9%(2015기준)로 급증했습니다.

    • 20세미만
      자궁경부암 진료인원 : 15명, 전체 암(여성)중 자궁경부암 비중 : 0.2%
    • 20세~29세
      자궁경부암 진료인원 : 2,194명, 전체 암(여성)중 자궁경부암 비중 : 11.9%
    • 30세~39세
      자궁경부암 진료인원 : 11,253명, 전체 암(여성)중 자궁경부암 비중 :14.9%
    • 40세~49세
      자궁경부암 진료인원 : 15,670명, 전체 암(여성)중 자궁경부암 비중 :9.2%
    • 50세~59세
      자궁경부암 진료인원 : 14,141명 전체 암(여성)중 자궁경부암 비중 : 5.9%
    • 60세~69세
      자궁경부암 진료인원 : 8,188명, 전체 암(여성)중 자궁경부암 비중 : 5.0%
      • 70세 이상
      자궁경부암 진료인원 : 5,131명, 전체 암(여성)중 자궁경부암 비중 : 3.6%

    ▲ 자궁경부암 연령별 진료인원 및 비중 (2015년) (출처:보험심사평가원)
